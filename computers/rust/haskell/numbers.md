# Math Functions
succ/pred -> Return num +/- 1
odd/even -> Return true if num is odd/even
round/floor/ceiling/truncate
\*\* -> Result can be a float
fromIntegral a -> Returns a as an Integral
negatve a -> Return negative value of a
