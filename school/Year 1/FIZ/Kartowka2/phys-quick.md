- a dot b = |a| \* |b| \* cos(a, b)

Iloczyn Wektorowy
- a x b = -b x a
- a x b = c
- c is perpendicular to the plane made by a, b

a (acceleration) = -gt<sup>2</sup> / 2 (if falling down)
