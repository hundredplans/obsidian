- f'(x<sub>0</sub>) < 0 => Funkcja maleje w punkcjie x<sub>0</sub>
- f'(x<sub>0</sub>) > 0 => Funkcja rosnie w punkcie x<sub>0</sub>
- Jezeli istnieje ekstremum lokalne to f'(x<sub>0</sub>) = 0 (Ale nie zawsze jezeli f'(x) = 0)

# L'Hopital
- Jezeli granica jest 0 / 0 lub ∞ / ∞
- Can keep the limit and do a derivative
- Sometimes have to do g(x) / 1 / f(x) to put it in a good form
- Remember sin(x) / nx = 1 / n
- sin(pi/2) = 1, cos(pi / 2) = 0
- sin(pi/6) = 1/2, cos(pi/6) = sqrt(3) / 2
- sin(pi/4), cos(pi/4) = sqrt(2) / 2
